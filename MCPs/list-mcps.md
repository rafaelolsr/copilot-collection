# 🧰 MCP Tools Stack

A curated stack of MCP tools for research, scraping, automation, and development workflows.

---

## 🔍 Search & Research

<details>
<summary><strong>View tools</strong></summary>

Tools focused on finding high-quality, relevant information.

- 🧠 **Exa MCP**  
  Semantic web search (great for deep research and non-obvious sources)

- ⚡ **Tavily MCP**  
  Fast, real-time search + summarized answers (AI-optimized, “Google-like”)

- 📚 **Context7 MCP**  
  Structured docs + version-aware references (ideal for dev documentation)

- 🗂️ **Ref MCP**  
  Internal knowledge retrieval (company docs, specs, private data)

**👉 Typical combo:**  
`Exa + Tavily + Context7` → powerful AI research stack

</details>

---

## 🌐 Web Scraping & Data Extraction

<details>
<summary><strong>View tools</strong></summary>

Tools that pull structured data from websites.

- 🔥 **Firecrawl MCP**  
  Clean scraping + multi-page crawling

- 🧪 **Chrome DevTools MCP**  
  Live browser inspection + manual/interactive scraping

**👉 Key difference:**

- **Firecrawl** → automated pipelines  
- **Chrome DevTools** → debugging + dynamic/manual extraction  

**👉 Typical combo:**  
`Firecrawl + Chrome DevTools` → robust scraping + debugging

</details>

## 🤖 Browser Automation & Testing

<details>
<summary><strong>View tools</strong></summary>
